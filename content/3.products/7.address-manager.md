# Address Manager

<img src="/img/address-manager-address.png" alt="Address Manager: an address" style="width:50%; border:solid 2px #eeeeee;" />

[KurrawongAI](https://kurrawong.ai)'s address data management tool.

It is designed to manage address data conforming to the [ANZ ICSM Address Model](https://linked.data.gov.au/def/addr) - the standard address data model for Australia & New Zealand.

### Features

* validation of all edits
    * using the Address Model's validator
* edit approvals workflow built-in
* ability to work with databases and message queues
* links Address data to other data, e.d. cadastral information, via Web Services
* manages Place Names & Road Names data too

### Status

Address Manager is in production use for Queensland within the 'QALI' system.

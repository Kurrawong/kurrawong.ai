# Address Manager

Address manager is one of [KurrawongAI](https://kurrawong.ai)'s [Address Services](https://address.dev.kurrawong.ai/) tools.

<img src="/img/address-manager-address.png" alt="Address Manager: an address" style="width:40%; border:solid 2px #eeeeee;" />

It is designed to manage address data conforming to the [ANZ ICSM Address Model](https://linked.data.gov.au/def/addr) - the standard address data model 
for Australia & New Zealand - based on the international address standard, [ISO 19160](https://committee.iso.org/sites/tc211/home/projects/projects---complete-list/iso-19160-1.html). 

::KCard{imgPosition="right" imgHalf}
We maintain a [copy of Address Manager online](https://address-demo.dev.kurrawong.ai), as configured for Queensland's [Spatial Information unit](https://www.data.qld.gov.au/article/news-and-events/earlier-open-data-events/govhack-2019-challenges/spatial-information). It contains real data from states & territories for testing. If you would like a 
walkthrough or access, please [get in touch](https://kurrawong.ai/about-us/get-in-touch).

### Features

* creation and editing of addresses
* validation of all edits
    * using the Address Model's validator
* edit approvals workflow built-in
* ability to work with databases and message queues
* links Address data to other data, e.d. cadastral information, via Web Services
* manages Place Names & Road Names data too
* scales to managing mullions of addresses

### Status

Address Manager is in production use for Queensland within the 'QALI' system and a test system is online, as above.

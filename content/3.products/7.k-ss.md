# K-SS (our own KurrawongAI Solid Server system)


<!-- describes a common problem-->

Personal data is increasingly concentrated in mass data stores that provide limited accessibility or control of access to those same individuals. Individuals, academics and data professionals are increasingly pushing for solutions that provide self-managed access for personal data. There are limited options for access control systems for Australians, particularly for large scale, supported services.

<!-- this provides a simple summary of the product -->

K-SS is KurrawongAIs access control system. Built on Linked Data for personal Data Management that creates the ability to establish surety of data provenance and source, while putting data ownership back into the control of the individual.

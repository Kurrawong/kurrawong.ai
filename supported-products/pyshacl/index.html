pySHACL

Products > Supported Products > pySHACL

Supported Product

pySHACL is a free and open source Python tool that validated RDF data using the Shapes Constraints Language (SHACL) standard.

Kurrawong staff helped develop and continue to assist maintaining pySHACL.

We provide the following services for pySHACL:

  • bug fixing
  • integration
    • into other Python tools such as workflows
  • extension
    • new tool functionality, performance improvements
  • validator creation
    • creation of SHACL shapes to validate your RDF data with, using pySHACL
  • training
    • workshops, cde examples and documentation for pySHACL’s use

Note that we provide free to use online instance of pySHACL within our RDFTools application that comes pre-loaded with many useful validators.
